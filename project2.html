NYC Taxi Random Forest Classifier

Using the NYC public taxi data, the project analyzes data from the three different types of rides (yellow taxis, green taxis, and for-hire rides) and uses a random forest classifiers to predict which borough a ride will end in. I used data from January 2018 for each ride type. The yellow taxi model performs at about 85% accuracy, the green taxi model performs at about 80% accuracy, and the for-hire rides perform at about 65-70% accuracy.
